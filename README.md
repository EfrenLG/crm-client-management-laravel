# CRM Clientes - Laravel

Aplicación web desarrollada con Laravel para la gestión de clientes.

## Funcionalidades

- Autenticación de usuarios (login/registro)
- Control de acceso por roles (admin)
- CRUD completo de clientes
- Validación de formularios
- Búsqueda por nombre/email
- Paginación de resultados

## Tecnologías

- PHP (Laravel)
- MySQL
- Blade
- Bootstrap

## Instalación

1. Clonar repositorio
2. composer install
3. npm install
4. configurar .env
5. php artisan migrate
6. php artisan serve
